Abbreviations

Prefixes of unquotees imply the following:

  • name: Name, tname: TermName, tpname: TypeName
  • value: T where T is value type that corresponds to given literal (e.g. Int, Char, Float etc)
  • expr: Tree an expression tree
  • tpt: Tree a type tree
  • pat: Tree a pattern tree
  • defn: Tree a definition tree
  • earlydefn: Tree an early definion tree (val or type definition)
  • self: Tree a self definition tree (i.e. val definition)
  • stat: Tree a statement tree (definition, expression or an import)
  • topstat: Tree a top-level statement tree (class, trait, package, package object or import)
  • enum: Tree a for loop enumerator
  • param: Tree a value parameter tree (i.e. val definition)
  • tparam: Tree a type paremeter tree (i.e. type definition)
  • parent: Tree a template parent
  • sel: Tree an import selector tree

Whenever a name has suffix s it means that it is a List of something. ss means List of Lists. So for example exprss means a List of Lists of expressions.
